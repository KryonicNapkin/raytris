# RayTris

Tetris clone written in C using [raylib](https://www.raylib.com/)

> [!NOTE]
> **Working!** :) But expect some bugs

## Installation

1. Clone this repository
```sh
git clone https://github.com/KryonicNapkin/raytris.git
```

2. Build, run and enjoy :)
```sh
cd raytris
make && ./raytris
```
